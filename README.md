# U.S. Inflation and Unemployment Analysis (2015–2023)

## Project Overview
This project uses Python to analyze U.S. inflation and unemployment trends based on real official economic data. It provides clear visualizations and insights for business, finance, and economic analysis users.

## Data Source
- Source: Federal Reserve Economic Data (FRED)
- Data type: Annual inflation rate & unemployment rate
- Period: 2015–2023
- Access date: 24 April 2026
- Website: https://fred.stlouisfed.org/

## Tools & Libraries
- Python 3
- Jupyter Notebook
- pandas
- matplotlib
- numpy

## Analysis Workflow
1. Data preparation
2. Data cleaning
3. Descriptive statistics
4. Visualization of trends
5. Insight generation

## Key Insights
- Inflation rose significantly in 2022.
- Unemployment spiked in 2020 and then recovered.
- The labor market remained stable in recent years.
- The data supports macroeconomic and financial decision-making.

## How to Run
1. Open the .ipynb file in Jupyter Notebook
2. Run all cells
3. View charts and results
